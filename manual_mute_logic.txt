Game audio follows:

manual override, if set
otherwise death_muted

Manual hotkey behavior:

- If game audio is currently unmuted, manual toggles to muted.
- If game audio is currently muted, manual toggles to unmuted.
- That makes manual a real override while death mute is active.

Defer mode:

- If `Manual defers to round` is ON, any manual override clears on round/score change.
- If it is OFF, the manual override stays in effect until the user toggles again.

Examples:

Mode       Starting state                 User presses manual     Game audio after press                On score change
Defer ON   Unmuted                        Force mute              Muted                                 Manual override clears
Defer ON   Death-muted                    Force unmute            Unmuted                               Manual override clears
Defer ON   Manual-muted                   Force unmute            Unmuted                               Manual override clears
Defer ON   Manual-unmuted over death      Force mute              Muted                                 Manual override clears
Defer OFF  Unmuted                        Force mute              Muted                                 Manual override stays
Defer OFF  Death-muted                    Force unmute            Unmuted                               Manual override stays
Defer OFF  Manual-muted                   Force unmute            Unmuted                               Manual override stays
Defer OFF  Manual-unmuted over death      Force mute              Muted                                 Manual override stays
